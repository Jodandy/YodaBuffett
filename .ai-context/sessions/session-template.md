## For AI Assistants: Using This Template

1. **Copy this template** to create a new session file with proper naming
2. **Fill out sections progressively** as you work, don't wait until the end
3. **Be specific with file references** - include paths and line numbers
4. **Update throughout the session** - the work log should reflect real-time progress
5. **Focus on handoff quality** - write for the AI assistant who comes after you
6. **Update related documentation** - keep domain docs and current-work.md in sync

### Key Success Patterns:
- **Specificity over generality** - concrete file paths, line numbers, error messages
- **Context preservation** - why decisions were made, not just what was done
- **Performance tracking** - actual metrics, not subjective assessments
- **Clear next steps** - actionable tasks that don't require re-research
